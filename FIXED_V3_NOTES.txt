eBay Image Tool fixed-v3
=========================

Core flow is preserved:
Gemini analyze -> OpenCV alpha-mask removal/cleanup -> add frame.

Important fixes:
- Gemini product_bbox is only a soft hint; it is never used as a hard crop.
- OpenCV keeps the original RGB pixels in the final cutout and only edits alpha.
- Source-image crop/trim is disabled by default to avoid cutting product edges.
- Improved alpha cleanup for grey/black listing borders and bottom disclaimer text.
- Improved protection for keep_regions so small screws/clips/brackets are retained.
- Frame composer keeps aspect ratio and searches more placements/scales.
- If no collision-free frame placement is possible, the frame is placed under the product as a safety fallback so it cannot cover the product.


V3.1 update:
- Giu nguyen flow Gemini -> OpenCV -> frame cua v3.
- Giam viền đen/con sót dọc mép bằng rebuild silhouette cho single-object mask.
- Giam xoa nen lem nhem / lỗ thủng nhỏ trong san pham bang cach fill cac lo alpha nho, nhung van giu cac lo cau truc lon (vi du center bore).


V3.2 update:
- Tightened Gemini analysis prompt to aggressively detect bottom disclaimers, tiny leftover text, and listing-border slivers.
- Added strict text-sweep retry when Gemini QA flags leftover_text in the final framed image.
- Still keeps the same closed flow: Gemini analyze -> OpenCV alpha cleanup -> add frame.


V3.3 open-source rembg engine:
- New default flow: Gemini analyze -> rembg AI alpha -> OpenCV alpha cleanup -> add frame.
- rembg is open-source and replaces OpenCV white-background threshold as the main cutout engine.
- OpenCV still only edits alpha mask; RGB remains original source image RGB.
- First run may download the selected rembg model. Keep internet on for first run.
- Install dependencies when running source mode: pip install -r requirements.txt
